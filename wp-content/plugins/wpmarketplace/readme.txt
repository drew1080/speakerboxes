WP Marketplace is a free full-featured Shopping Cart/eCommerce Plugin with everything you need on an easy UI, to build online shop or marketplace.

== Description ==

The WP Marketplace plugin is a full-featured WordPress Shopping Cart/e-commerce solution that is extremely easy to install and even easier to maintain. Transform your WordPress website into a full featured eCommerce store just in few click.

= All Features to build a complete online shop: =

* Easy administration
* PayPal Integrated
* User-role based discount management
* Integrated Feature Management and Pricing Table 
* Multiple shipping option (extendable) 
* Multiple payment gateways (extendable)
* Sales Tax
* Stock Management
* Product & Price variation ( like colors, sizes ... )
* Multilevel pricing
* Invoice generation
* Customizable email templates
* Stock control
* Global coupon option
* Product specific coupon option
* Featured products
* and many more...

= Special Features for Selling Digital Products: =

* Sale digital product securely
* Multi-file upload support with single product, download will come as a zip pack
* Download protection
* Product Demo option
* Licensing System - Protect product with license key
* and many more...

== Installation ==


1. Upload `wpmarketplace` to the `/wp-content/plugins/`  directory
2. Activate the plugin through the 'Plugins' menu in WordPress
