Enable your clients or theme users to easily change the color scheme of your design. With live preview!

== Description ==

This plugin adds an option page where you can add color options and the plugin hooks this elements to the Customizer where your client or theme users can customize the design.

Since 0.2 the custom designed panel to change colors is removed and now design tweaks are done through the WordPress Theme Customizer.

== Installation ==

1. Upload the whole plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto Appearance -> Color Manager and add elements (see screenshots)
4. Goto the Customizer and configure the colors
5. Enjoy!
